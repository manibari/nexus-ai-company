## 2. 組件規格

### 2.1 搜尋框 (StockSearch)

```
┌─────────────────────────────────────┐
│ 🔍 Search stocks (e.g., AAPL)      │
└─────────────────────────────────────┘
         ↓ (輸入時顯示)
┌─────────────────────────────────────┐
│ AAPL    Apple Inc.           equity │
│ AMZN    Amazon.com Inc.      equity │
│ AMD     Advanced Micro...    equity │
└─────────────────────────────────────┘
```

**互動規格：**
- 輸入 1 字元後開始搜尋
- 300ms 防抖延遲
- 最多顯示 10 筆結果
- Enter 選擇第一筆
- ESC 關閉下拉選單
- 點擊外部關閉

## 3. 色彩規範

| 用途 | 色碼 | 說明 |
|------|------|------|
| 背景 (主) | `#0f172a` | slate-900 |
| 背景 (次) | `#1e293b` | slate-800 |
| 背景 (卡片) | `#334155` | slate-700 |
| 文字 (主) | `#ffffff` | white |
| 文字 (次) | `#94a3b8` | slate-400 |
| 文字 (弱) | `#64748b` | slate-500 |
| 強調色 | `#06b6d4` | cyan-500 |
| 漲/正向 | `#22c55e` | green-500 |
| 跌/負向 | `#ef4444` | red-500 |
| 警告 | `#eab308` | yellow-500 |

## 5. 使用者流程

```
[首頁] → [搜尋股票] → [選擇股票] → [查看報價]
                                      ↓
                    ┌────────┬────────┼────────┬────────┐
                    ↓        ↓        ↓        ↓        ↓
                [K線圖] [指標] [基本面] [AI分析] [回測]
                    ↓        ↓        ↓        ↓        ↓
              [切換週期] [解讀] [評估] [建議] [執行回測]
```
