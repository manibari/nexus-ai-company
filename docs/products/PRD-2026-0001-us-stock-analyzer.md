## 3. 數據來源分析 (Data Sources Analysis)

### 3.1 免費 API 選項

| API | 數據類型 | 限制 | 適用場景 |
|-----|----------|------|----------|
| Yahoo Finance (yfinance) | 股價、財報 | 無官方 API，可能被封鎖 | MVP / 個人使用 |
| Alpha Vantage | 股價、技術指標 | 5次/分鐘 (免費) | 低頻需求 |
| Finnhub | 即時報價、新聞 | 60次/分鐘 (免費) | 新聞情緒 |

### 3.2 付費 API 選項

| API | 數據類型 | 月費 | 適用場景 |
|-----|----------|------|----------|
| Polygon.io | 即時股價、歷史 | $29+ | 專業級 |
| IEX Cloud | 股價、財報、新聞 | $9+ | 全方位 |
| Alpha Vantage Pro | 完整數據 | $50+ | 高頻需求 |
| Alpaca Markets | 交易 + 數據 | 免費 (交易) | 實際下單 |

### 3.3 PM 建議

根據不同預算，建議的數據方案：

| 預算等級 | 建議方案 | 月費 |
|----------|----------|------|
| 低 ($0-10) | yfinance + Finnhub | ~$0 |
| 中 ($10-50) | IEX Cloud Growth | ~$19 |
| 高 ($50+) | Polygon.io + IEX | ~$60+ |

## 4. 技術架構建議 (Technical Architecture)

### 4.1 Web App 架構

```
┌─────────────────────────────────────────────────────────────────┐
│                         Frontend                                 │
│  React + TailwindCSS + TradingView Lightweight Charts           │
└───────────────────────────────┬─────────────────────────────────┘
                                │
                                ▼
┌─────────────────────────────────────────────────────────────────┐
│                         Backend                                  │
│  FastAPI + Python + SQLite/PostgreSQL                           │
│                                                                  │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐             │
│  │ Stock API   │  │ Analysis    │  │ AI Module   │             │
│  │ Integration │  │ Engine      │  │ (Claude)    │             │
│  └─────────────┘  └─────────────┘  └─────────────┘             │
└───────────────────────────────┬─────────────────────────────────┘
                                │
                                ▼
┌─────────────────────────────────────────────────────────────────┐
│                    External Services                             │
│  ┌─────────┐  ┌─────────┐  ┌─────────┐  ┌─────────┐            │
│  │ Stock   │  │ News    │  │ Broker  │  │ AI/LLM  │            │
│  │ API     │  │ API     │  │ API     │  │ API     │            │
│  └─────────┘  └─────────┘  └─────────┘  └─────────┘            │
└─────────────────────────────────────────────────────────────────┘
```

### 4.2 核心模組

| 模組 | 功能 | 技術 |
|------|------|------|
| 行情模組 | 即時/歷史股價 | REST API + WebSocket |
| 圖表模組 | K線、技術指標 | TradingView Lightweight Charts |
| 分析模組 | 技術/基本面分析 | pandas, numpy, ta-lib |
| AI 模組 | 智能選股、對話 | Claude API |
| 交易模組 | 下單、持倉管理 | Alpaca API (選配) |
| 回測模組 | 策略回測 | backtrader / zipline |
